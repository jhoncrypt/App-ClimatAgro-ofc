# **App Name**: MeteoViz

## Core Features:

- Data Fetch: Fetch weather data from the Open-Meteo API using the provided URL.
- JSON Conversion: Parse the fetched JSON data.
- Interactive Table: Display the weather data in an interactive and sortable table format.
- Data Visualization: Implement chart visualizations of daily temperature range, precipitation and shortwave radiation based on user preferences.
- Data export: Add capability to export the displayed table data to Excel.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to reflect the sky and weather theme.
- Background color: Light gray (#F0F8FF) for a clean and readable interface. 
- Accent color: Soft orange (#FFB347) for highlighting important data and interactive elements.
- Body and headline font: 'PT Sans' for a modern and readable style, suitable for displaying weather data.
- Use weather-related icons to represent different parameters, such as temperature, precipitation, and wind speed.
- Use a clear and structured layout with intuitive data groupings for easy understanding.
- Implement subtle animations to highlight changes in weather parameters, making the interface dynamic.
